Movies API Documentation


Objective

To build a movies api that allows users to browser movie selections and obtain information about the movie such as: genre, description, actors/actresses, and ratings.

Endpoints

  • GET Request Endpoints
    • /movies
      • Returns json data on all movies
        • {        
    title: 'Goodwill Hunting',
    year: 1997,
    genre: ['Drama', 'Romance'],
    director: {
        name: 'Gus Van Sant',
        bio: 'Gus Van Sant was born on...etc
        dob: 1952-7-24,
        dod: 'alive'
    }
},
...
    • /Movies/:title
      • Reutrns json data on a single movie
        • {        
    title: 'Goodwill Hunting',
    year: 1997,
    genre: ['Drama', 'Romance'],
    director: {
        name: 'Gus Van Sant',
        bio: 'Gus Van Sant was born on...etc
        dob: 1952-7-24,
        dod: 'alive'
    }
}
    • /genre/:genre
      • Returns json data about a genre
    • /directors
      • Reutnrs a list of all directors
    • /directors/:director
      • Returns json data about a single director
  • POST Requests
    • /registration
      • Registers new user
    • /:userID/favorites/add/:movie
      • Allows user to add a movie to their favorites list
  • PUT Requests
    • /remove_acct/:userID
      • Allows user to update thier user information
  • DELETE Requests
    • /:userID/favorites/delete/:movie
      • Allows a user to delete a movie from their favorites list
    • /remove_acct/:userID
      • Allows a user to remove their account from the app
