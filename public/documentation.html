MyFlix API Documentation

Objective

To build a server-side component of a movie API that allows access to basic movie information: description, year of release, and director information.

                                        {
                                            "name": "Adventure"
                                        },
                                        {
                                            "name": "Thriller"
                                        }
                                    ],
                                    "directorInfo": [
                                        {
                                            "name": "Steven Spielberg",
                                            "bio": "One of the most..."
                                        }
                                    ]
                                },
                                ...

GET /genres

GET /genres/:genre

Descrption: Returns a list OR a single document about genres in the database

Response Example:

                                    "_id": "6189812da64de87fdf156f69",
                                    "name": "Action",
                                    "description": "Action film is a film genre..."
                                },
                            ]

GET /director

GET /director/director_fullname

Descrption: Returns list or a single docuemnt about director information

Response Example:
                        
                            {
                                "_id": "618804104c56905de44cbcff",
                                "name": "Steven Spielberg",
                                "bio": "One of the most influential...",
                                "birthday": "1970-01-01T00:00:01.916Z",
                                "deathday": null
                            }

POST /registration

Descrption: Creates user if user credentials not in the database.

Body:
                        
                            {
                                "username": "test1",
                                "password": "test1password",
                                "email": "test1@yahoo.com",
                                "birthday": "1971-5-05"
                            }
Response Example:
                        
                            {
                                "username": "test1",
                                "password": "test1password",
                                "email": "test1@yahoo.com",
                                "birthday": "1971-05-05T05:00:00.000Z",
                                "favoriteMovies": [],
                                "_id": "61a6e3c35d6cd666fdd84fb3",
                                "__v": 0
                            }

POST /user/:userID/favorites/:movie_title

Descrption: Adds a movie to the user's favorites list

Response Example:
                        
                            {
                                "_id": "61a6e3c35d6cd666fdd84fb3",
                                "username": "test1",
                                "password": "test1password",
                                "email": "test1@yahoo.com",
                                "birthday": "1971-05-05T05:00:00.000Z",
                                "favoriteMovies": [
                                    "6186f8a74c56905de44cbcfb"
                                ],
                                "__v": 0

PUT /user/:userID/update

Descrption: Update's user information. All required fields must be send in body.

Body:
                        
                            {
                                "username": "test123456", (required)
                                "password": "test1password", (required)
                                "email": "test1@yahoo.com", (required)
                                "birthday": "1971-5-05" (required)
                            }
Response Example:
                        
                            {
                                "_id": "61a6e3c35d6cd666fdd84fb3",
                                "username": "test123456",
                                "password": "test1password",
                                "email": "test1@yahoo.com",
                                "birthday": "1971-05-05T05:00:00.000Z",
                                "favoriteMovies": [
                                    "6186f8a74c56905de44cbcfb"
                                ],
                                "__v": 0
                            }

DELETE /:userID/favorites/delete/:movie_title

Descrption: Adds a movie to the user's favorites list.

Response Example:
                        
                            (movie title) has been removed from user's favorites list

DELETE /remove/:userID

Descrption: Deletes user from database.

Response Example:
                        
                            Successful: (userID) user has been successfully removed
                            Unsuccessful: (userID) user was not found
