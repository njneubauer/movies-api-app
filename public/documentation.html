Movies API Documentation


Objective

To build a server-side component of a movie API that allows access to basic movie information: description, year of release, and director information.

Endpoints

  • GET Request Endpoints
    • /movies
      • Returns json data on all movies
      • Response Example:
        • {
    "title": "Jaws",
    "year": 1975,
    "genre": [
        "Adventure",
        "Thriller"
    ],
    "director": {
        "name": "Steven Spielberg",
        "bio": "One of the most influential personalities...etc,
        "dob": 1916,
        "dod": "alive"
    }
},
...
    • /movies/:title
      • Reutrns json data on a single movie
      • Response Example:
        • {
    "title": "Jaws",
    "year": 1975,
    "genre": [
        "Adventure",
        "Thriller"
    ],
    "director": {
        "name": "Steven Spielberg",
        "bio": "One of the most influential personalities...etc,
        "dob": 1916,
        "dod": "alive"
    }
}
    • /genre/:genre
      • Returns json data about a genre
      • Response Example:
        • {
    "title": "Jaws",
    "genre": [
        "Adventure",
        "Thriller"
    ],
}
    • /directors
      • Reutnrs a list of all directors
      • Response Example:
        • "directors": [ "Ethan Coen", "Gus Van Sant", "Milos Forman", "Quentin Tarantino", ]
    • /directors/:director
      • Returns json data about a single director
      • Response Example:
        • "director": {
    "name": "Steven Spielberg",
    "bio": "One of the most influential personalities...etc,
    "dob": 1916,
    "dod": "alive"
}
  • POST Requests
    • /registration
      • Registers a new user
      • Responds with text that a "new user has been added!"
    • /:userID/favorites/add/:movie
      • Allows user to add a movie to their favorites list
  • PUT Requests
    • /remove_acct/:userID
      • Allows user to update thier user information
  • DELETE Requests
    • /:userID/favorites/delete/:movie
      • Allows a user to delete a movie from their favorites list
    • /remove_acct/:userID
      • Allows a user to remove their account from the app
